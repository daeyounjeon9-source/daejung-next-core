DAEJUNG NEXT V2 3개 수정 보고서 - 2026-06-10

수정 기준 파일: DA2(1).zip
수정 범위: V2 UI/편성표/인트로만 수정. 라우터/검색/상품데이터/결제 로직은 변경하지 않음.

1) 장바구니 위치 보정
- 하단 플로팅 장바구니가 상품카드 위를 덮는 문제 보정.
- 방향키와 균형 맞도록 오른쪽 하단으로 낮춰 고정.
- 모바일에서 방향키는 왼쪽, 장바구니는 오른쪽으로 분리.

2) 방송일정표 요일별 구조 보정
- 위쪽 큰 LIVE 편성표는 숨김 처리.
- 하단 흐르는 편성표 줄만 기본 노출.
- [편성표] 버튼 클릭 시 패널 오픈.
- 패널 내부: 지난 1주일 / 오늘 / 다음 1주일 탭 구성.
- 지난 1주일과 다음 1주일은 월/화/수/목/금/토/일 요일별 편성.
- 각 일정에는 시간, 상품명, 채널, 가격, 상태, 🔔 알림, 바로주문 포함.
- 바로주문 클릭 시 4차방 구매결정 쇼룸(/decision-room) 이동.

3) 인트로 복구
- localStorage 기준 때문에 인트로가 계속 사라지는 문제 완화.
- sessionStorage 기준으로 변경하여 새 세션 첫 접속 시 1회 표시되도록 조정.

검수
- npm install --include=optional 실행 후 빌드 환경 정리.
- npm run build 성공.
