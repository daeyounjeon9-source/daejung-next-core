DAEJUNGNEXT_V3 검수/수정 보고서

기준 파일: v22011.zip
작업명: DAEJUNGNEXT_V3

1. 파일 정리
- 원본은 직접 수정하지 않고 작업본에서 정리.
- node_modules 제거 대상: 재설치 가능 의존성.
- dist 제거 대상: 빌드 결과물.
- 최종 ZIP에는 node_modules/dist 미포함.

2. 명칭 정리
- 사용자 화면에서 개발용 방 명칭을 서비스 명칭으로 교체.
- 1차방/쇼케이스 계열 → 선택된 자 6
- 2차방 → 특별관
- 3차방 → 전체상품관
- 4차방 → 구매결정 쇼룸

3. --- 메뉴 정리
- 메뉴 안 영어 표기 제거.
- 유지: 공지사항, 회원가입, 상품 판매 및 홍보 신청, 관리자 로그인, 회원탈퇴.
- 제거 대상: 구매자 회원가입, 판매자 회원가입, 주문조회, 홍보관, 최근본상품, 관심상품, 장바구니, 고객센터, 약관류 직접 노출.

4. 메뉴 UI 보강
- 모바일에서 반쪽만 보이는 문제 완화.
- 메뉴 가로폭 확대.
- 버튼 높이/가로 패딩/세로 간격/문단 간격 확대.
- 한글 중심으로 가독성 우선.

5. 상품 판매 및 홍보 신청
- 상품 판매 및 홍보 등록센터 유지.
- 입력, 임시저장, 수정, 복사, 삭제, 재등록 흐름 확인.
- 판매자라는 표현보다 판매·홍보 활동 신청 중심으로 정리.

6. 빌드 검수
- npm install 후 npm run build로 확인.
- 실제 브라우저 전체 클릭 검수와 휴대폰 실기기 검수는 사용자 PC에서 추가 확인 필요.
