DAEJUNG NEXT V3004 전면 수정 보고서

수정 완료 항목
1. 공지사항 메뉴
- 기존: 공지사항 클릭 시 페이지 이동 없이 아래로 스크롤
- 수정: /notice 독립 공지사항 페이지로 이동

2. 로그인/회원가입 오류 문구
- 회원가입 중복 이메일: "이미 가입된 이메일입니다."
- 가입되지 않은 이메일 로그인: "가입되지 않은 이메일입니다."
- 비밀번호 오류: "비밀번호가 올바르지 않습니다."
- 관리자 권한 없는 계정: "관리자 권한이 없는 계정입니다."
- 판매자 권한 없는 계정: "판매자 계정이 아닙니다. 판매자 회원가입 또는 승인을 확인해주세요."

3. 판매자 회원가입 전면 용어 정리
- 포스틱 제거 확인
- 정산계좌 단일 입력 제거
- 은행명 / 계좌번호 / 예금주 분리
- 교환학생 주소 제거
- 교환·반품 주소로 정리
- 사업자등록증 파일명 제거
- 사업자등록증 업로드로 변경
- 파일 선택 input 적용
- 상호명/브랜드명, 업종/업태, 고객센터 연락처로 정리

4. 판매자 유형 추가
- 일반 판매자
- 농수산물 생산자
- 소상공인
- 중소기업
- 기업/브랜드
- 통신판매업 신고번호는 "해당 시"로 변경

5. 한글 표현 정리
- THE CHOSEN SIX 제거
- 선택된 자 6 일부 표현을 쇼케이스 상품관으로 정리

검수 결과
- npm install 성공
- npm run build 성공
- build 경고: chunk size warning만 존재, 실행 오류 아님

휴대폰 검수 절차
1. npm run dev -- --host
2. ipconfig에서 IPv4 확인
3. 휴대폰 브라우저에서 http://IPv4주소:5173 접속
