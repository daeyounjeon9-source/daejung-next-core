DAEJUNG NEXT V2 LIVE 편성표 정리 패치

적용 기준: 사용자가 업로드한 DA2(1).zip
작업 원칙: 압축 안에 또 V2 폴더를 넣지 않고, 현재 프로젝트 루트 그대로 수정

수정 내용:
1. 방향키 복구
- 검은색/대형 하단 방향키 override를 마지막 CSS에서 다시 덮어쓰기
- 화이트 글래스 + 블루 톤의 작고 낮은 방향키로 복구

2. 위쪽 큰 LIVE 편성표 제거
- live-schedule-board 화면 노출 제거
- LIVE 카드 아래를 과도하게 차지하던 1900년대식 큰 표 제거

3. 밑줄 편성표로 통합
- 하단 영역은 [📅 편성표] 하나만 노출
- 미리주문/빠른주문/바로주문 외부 버튼 제거
- 편성 항목은 가로 슬라이드로 흐르게 유지

4. 편성표 내부 기능
- 편성표 터치 시 패널 열림
- 지난 1주일 / 오늘 / 다음 1주일 탭
- 각 방송 항목에 시간, 채널, 상품명, 가격, 상태 표시
- 🔔 알림: 고객이 눌러두면 방송 시간 알림 설정 토스트 표시
- 바로주문: 4차방 구매결정 쇼룸(/decision-room) 이동

검수:
- npm install --include=optional 실행
- npm run build 성공
